# Security Policy

忙个明白是本地桌面工具，默认不会上传工作记录或截图。

## Supported Versions

当前公开版本仍处于早期 Alpha 阶段。安全修复会优先覆盖最新公开版本。

## Reporting a Vulnerability

如果你发现安全问题，请不要在公开 Issue 中披露细节。请通过 GitHub 的私有安全报告入口提交：

https://github.com/czcyolo/busy-clear/security/advisories/new

报告时请尽量包含：

- 受影响版本
- 操作系统版本
- 复现步骤
- 可能影响的数据或权限范围

## Privacy Reminder

提交截图、录屏或日志前，请先检查是否包含真实工作内容、客户信息、个人隐私或敏感文件路径。
