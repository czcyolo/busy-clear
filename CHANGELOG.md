# 更新日志

所有重要版本变更记录在这里。

## v0.1.0 - Mac MVP

第一阶段 MVP，用于 macOS 本机体验。

### 已完成

- 添加、开始、停止和继续主任务。
- 主任务计时累积展示。
- 同一时间只允许一个主任务运行。
- 主任务运行中截图并创建子项。
- 子项运行时可结束并回到主线。
- 右键截图点追加同色后续。
- 拖动截图点到垃圾桶删除子项或某次后续。
- 支持主任务快捷键和截图快捷键。
- 支持菜单栏入口、前三个主任务快速切换。
- 支持无感模式和置顶。
- 支持主任务拖动排序。
- 支持导出过去 24 小时时间轴 PNG。
- 截图和导出文件保存到用户图片目录下的日期文件夹。

### 已知限制

- 当前 Mac 安装包未做 Apple Developer ID 签名。
- Windows 版本尚未正式适配和验证。
- 时间轴导出用于个人复盘和沟通，不是不可篡改审计证据。
