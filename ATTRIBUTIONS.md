# 第三方素材说明

这个文件用于记录公开发布时需要说明来源的素材。

## 应用源码

应用源码使用的主要开源依赖包括 Electron、React、Vite、TypeScript、Vitest、lucide-react 等。具体版本以 `package.json` 和锁定文件为准。

## 应用图标和界面素材

应用图标、菜单栏图标和界面内的牛形象素材为项目素材。

公开发布前请再次确认这些素材均可用于公开展示和分发。

## 宣传视频

宣传视频用于展示“忙个明白”的产品理念和核心操作，不属于软件运行所必需的部分。

当前最终候选视频使用了以下外部素材：

- BGM：Mixkit - Digital Clouds
- BGM 来源：https://assets.mixkit.co/music/175/175.mp3
- Mixkit License：https://mixkit.co/license/

公开发布前建议再次核对 Mixkit 的最新许可说明，并在 GitHub Release 或视频说明中保留本文件链接。

## 说明

如果后续新增字体、音乐、音效、图片、图标或视频素材，请把来源、许可和使用位置补充到这个文件中。
