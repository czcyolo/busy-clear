# 忙个明白 使用说明

“忙个明白”用来快速记录一天做了什么：开始任务、被打断时截图、最后导出一张时间轴。

## 按钮说明

`+` 添加一个主任务。

`开始 / 结束` 开始或停止当前主任务。再次开始同一个任务时，时间会继续累积。开启另一个主任务时，之前运行中的主任务会自动停止。

右键 `开始 / 结束` 设置主项快捷键。没有任务运行时，快捷键会开启列表里的第一个主任务；有任务运行时，快捷键会停止当前主任务。

`相机` 在当前主任务下截图，并自动创建一个子项开始计时。

`方块` 结束当前任务或当前截图子项。

`垃圾桶` 删除主任务。把时间条上的截图点拖到垃圾桶，可以删除对应子项或某次后续。

`无感开关` 开启后，应用不会出现在 Dock 和 `Command+Tab` 切换列表里；重新打开应用后会默认关闭。

`图钉` 让窗口保持在普通窗口上方。

`下载` 导出过去 24 小时的时间轴 PNG。

右上角菜单栏尾巴图标也可以打开说明书、切换无感模式、切换置顶或退出应用。

菜单栏会显示前三个主任务。点击某个任务会开始计时；再次点击正在运行的任务会停止计时。

右键 `相机` 设置截图快捷键。没有子项运行时，快捷键会截图并创建子项；子项运行时，快捷键会结束当前子项。按 `Esc` 取消，按退格清除。

右键主任务的 `垃圾桶` 打开截图文件夹。

右键时间条上的截图点，为同一个子项追加后续。

拖动主任务栏空白区域或时间段，可以调整主任务顺序。鼠标停在时间段上仍然可以查看时间区间明细。

## 日常使用

1. 点击 `+` 添加主任务，写上任务名称。
2. 点击 `开始`，开始记录这项工作。常用任务可以先拖动排序，排在前面的任务会显示在菜单栏里。
3. 被打断时点击 `相机`，软件会截图并创建子项。
4. 子项处理完后点击 `方块`，结束这次打断。
5. 一天结束后点击 `下载`，导出过去 24 小时的时间轴。

## 文件位置

截图和导出的时间轴保存在：

```text
~/Pictures/忙个明白/截图/YYYY-MM-DD/
```

截图文件名类似：

```text
数据分析10：15.png
```

工作记录保存在本机，不会上传云端。

## macOS 权限

首次截图时，macOS 可能会要求授予“屏幕录制”权限。

如果截图失败，请到：

```text
系统设置 -> 隐私与安全性 -> 屏幕录制
```

允许“忙个明白”后，重新打开应用再试。

## 注意

这个工具适合个人复盘、写日报、和沟通“今天忙了什么”。它不是不可篡改的审计系统，也不建议保存敏感隐私截图。
